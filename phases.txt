PHASE 1: Project Setup & Core Infrastructure

Initialize Next.js 16 project

npx create-next-app@latest hrms --ts
cd hrms


Install core dependencies

npm install tailwindcss shadcn-ui prisma @prisma/client better-auth react-query zod pusher-js


Setup Tailwind & Shadcn

npx tailwindcss init -p
# Configure shadcn
npx shadcn-ui init


Setup Prisma + Neon

Add DATABASE_URL in .env

Add full schema (from final schema) to prisma/schema.prisma

Run migration:

npx prisma migrate dev --name init_hrms
npx prisma generate


Setup BetterAuth

Configure with User table

Roles: ADMIN, HR, EMPLOYEE

Add role-based middleware for route protection
